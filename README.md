#  Videojuego basado en la saga Souls

Este es un videojuego que se inspira en la saga Souls, tanto en la tematica, la ambientacion y mecanicas principales 



---

##  Tecnologías principales

- Motor: Unity 

- Lenguajes: C#
 

---

##  Características

- Menu principal
- Selector de personajes
- Nivel1, Nivel2 y Nivel 3
- IA de enemigos basada en estados de maquina finitos 
- sistema de particulas e iluminacion
